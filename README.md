# 📁 Landings — Estructura del Proyecto

Colección de páginas landing independientes. Cada página tiene su propia carpeta con su archivo HTML, hoja de estilos CSS y subcarpeta de imágenes.

---

## 🗂️ Estructura General

```
Landings/
├── index.html             ← Menú principal (punto de entrada)
├── page3/
├── page4/
├── page6/
├── page10/
├── page11/
├── page12/
├── page14/
├── page15/
├── page22/
├── page33/
├── page38/
├── page40/
├── page41/
├── page45/
└── page50/
```
